# Inventory Health Analytics Dashboard

## Overview
Developed an interactive Power BI dashboard to monitor inventory performance and analyze key inventory metrics.

## Tools Used
- Power BI
- Microsoft Excel

## Features
- KPI Cards (Total Units Sold, Total Stock Available, Average Fill Rate)
- Store-wise Sales Analysis
- Sales by Category
- Inventory Health Distribution
- Sales Trend Analysis
- Interactive Store and Category Filters

## Business Insights
- Compared sales across stores.
- Analyzed inventory health distribution.
- Identified sales trends over time.
- Enabled interactive filtering for better decision-making.
